## Getting Started

### Prerequisites

- check npm version
  ```sh
  npm -v
  ```
- check node version
  ```sh
  node -v
  ```

If you do not have those installed, download [here](https://nodejs.org/en/).

### Installation

Install npm dependencies

```sh
npm install
```
Check truffle version

```sh
truffle version
```

If truffle is not recognized use the following command

```sh
./node_modules/.bin/truffle version
```
